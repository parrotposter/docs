# Удаление поста

Эта мутация позволяет удалить пост. После удаления, сервис также удалит пост из соцсетей, если тот был опубликован.

::: warning Важно
После удаления пост не может быть восстановлен.
:::

## Пример запроса

```graphql
mutation {
  deletePost(id: "post_id_12345") {
    id
  }
}
```

## Параметр `id`

Уникальный идентификатор поста, который нужно удалить.

## Ответ `deletePost`

Мутация возвращает объект удаленного поста. Вы можете запросить любые поля поста в зависимости от ваших потребностей. Посмотрите [структуру объекта Post](./item.md).
