# Структура Post

Этот объект представляет собой пост, который был создан в Parrotposter.

### Поля Post

| Поле        | Тип       | Описание                                          |
| ----------- | --------- | ------------------------------------------------- |
| `id`        | ID        | Уникальный идентификатор поста                    |
| `status`    | String    | Статус поста (`PREPARE`, `SUCCESS`, `READY`, etc) |
| `fields`    | Object    | Содержимое поста (те же поля, что и при создании) |
| `networks`  | Object    | Информация о том, куда публиковать                |
| `publishAt` | String    | Запланированное или фактическое время публикации  |
| `results`   | [Object]? | Результаты публикации                             |
| `createdAt` | String    | Время создания поста                              |
| `updatedAt` | String    | Время последнего обновления                       |

### Статус поста

| Статус       | Описание                                                   |
| ------------ | ---------------------------------------------------------- |
| `PREPARE`    | Подготовка к публикации                                    |
| `READY`      | Готово к публикации, ожидает запланированного времени      |
| `SUCCESS`    | Публикация завершена успешно                               |
| `FAIL`       | Публикация не удалась, возникла ошибка в одной из соцсетей |
| `PUBLISHING` | Публикация выполняется                                     |
| `UPDATING`   | Обновление поста                                           |
| `DELETING`   | Удаление поста                                             |

### Поле `fields`

Объект, описывающий текстовое содержимое и медиа.

| Поле        | Тип       | Описание                           |
| ----------- | --------- | ---------------------------------- |
| `text`      | String?   | Текст поста                        |
| `link`      | String?   | Ссылка для прикрепления к посту    |
| `tags`      | String?   | Теги (хештеги)                     |
| `imageIds`  | [ID]?     | Список идентификаторов изображений |
| `images`    | [Object]? | Список изображений                 |
| `fileIds`   | [ID]?     | Список идентификаторов файлов      |
| `files`     | [Object]? | Список файлов                      |
| `neetUtm`   | Boolean?  | Прикрепить UTM-метки               |
| `utmParams` | Object?   | UTM-метки                          |
| `extra`     | Object?   | Дополнительные параметры           |

### Поле `fields.extra`

Дополнительные параметры для специфичных настроек соцсетей.

| Поле          | Тип      | Описание                                                                                     |
| ------------- | -------- | -------------------------------------------------------------------------------------------- |
| `vkFromGroup` | Boolean? | Публиковать от имени группы (`true`) или автора (`false`) во ВКонтакте. По умолчанию: `true` |
| `vkSigned`    | Boolean? | Добавлять подпись с автором поста во ВКонтакте. По умолчанию: `false`                        |

### Поле `networks`

Объект, определяющий, куда публиковать пост.

| Поле       | Тип      | Обязательное | Описание                                                                            |
| ---------- | -------- | ------------ | ----------------------------------------------------------------------------------- |
| `accounts` | [String] | Обязательно  | Список идентификаторов аккаунтов, полученных из [listSocialAccounts](../socials.md) |

### Поле `results`

Список результатов публикации по соцсетям.

| Поле               | Тип     | Описание                            |
| ------------------ | ------- | ----------------------------------- |
| `accountId`        | String  | Идентификатор аккаунта              |
| `success`          | Boolean | Успешность публикации               |
| `linkToSocialPost` | String? | Ссылка на пост, созданный в соцсети |
| `errorMsg`         | String? | Сообщение об ошибке                 |
